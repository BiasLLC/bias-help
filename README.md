# BIAS Writer Help Site

Documentation site for BIAS Writer, deployed at **help.bias.pub**

## Setup

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Deployment

Deployed automatically to Vercel when pushing to `main` branch.

**Domain:** help.bias.pub

## Content

The user guide content is in `content/user-guide.md` - copy your USER_FEATURES.md file there.

## Structure

- `app/layout.tsx` - Main layout with header, sidebar, footer
- `app/page.tsx` - Home page that renders the markdown
- `app/globals.css` - Global styles and markdown styling
- `content/user-guide.md` - The actual USER_FEATURES.md content
